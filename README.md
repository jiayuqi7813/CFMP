# CFMP Crossfire-Music-Player
基于CNN分类器的穿越火线多人生化捡了个音响

# 算法
考虑到一些问题，模型训练算法暂不开源，模型识别问题请提issue

# 用法
版本：python3.10

`pip3 install -r requirements.txt`

启用obs虚拟摄像头

然后python3 app.py即可，分辨率相关请自行调整。
